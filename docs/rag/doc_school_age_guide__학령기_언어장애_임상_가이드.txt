# 학령기 언어장애 임상 가이드

## 개요

학령기(만 6~12세)는 언어가 학습의 도구로 전환되는 시기다. 취학 전에 언어 발달 문제가 해결되지 않으면 학령기에 읽기, 쓰기, 작문, 수업 이해, 또래 관계에서 어려움으로 이어진다. 학령기 언어재활은 단순 언어 발달 촉진이 아니라 **학업 언어(academic language)** 역량과 **메타언어(metalinguistic) 능력** 발달에 초점을 맞춘다.

학령기 아동에서 임상적으로 가장 중요한 진단 범주는 **발달성 언어장애(Developmental Language Disorder, DLD)**의 학령기 지속과 **언어기반 학습장애(Language-Based Learning Disabilities, LBLD)**다.

---

## 취학 전 vs 학령기 언어재활의 차이

| 구분 | 취학 전 | 학령기 |
|---|---|---|
| 언어 목적 | 일상 의사소통 발달 | 학습 도구로서의 언어 |
| 핵심 영역 | 어휘, 구문, 화용, 음운 | 읽기, 쓰기, 구어 담화, 학업 어휘 |
| 주요 지표 | MLU, NDW, TTR, PCC | 읽기 정확도·유창도, 작문, 이야기 구조 |
| 중재 환경 | 치료실, 가정 | 치료실 + 교실 연계 |
| 협력 대상 | 보호자 | 보호자 + 담임교사 + 특수교사 |

## 발달성 언어장애 (DLD)의 학령기 양상

### 정의

DLD는 알려진 신경학적·감각적·지적 원인 없이 언어 발달에 지속적인 어려움을 보이는 상태다(Bishop 외, 2017). 취학 전에 진단되기도 하지만, 학령기에 처음 진단되는 경우도 많다.

### 학령기 주요 증상

**구어 영역**
- 문장 반복(비단어 따라 말하기) 어려움
- 복잡한 구문 이해 오류 ("책을 읽은 후에 나한테 주세요" 같은 시제·조건절)
- 이야기 구성 능력 저하 (시간 순서, 원인-결과 관계)
- 주제 유지와 대화 전략 사용 어려움

**읽기 영역**
- 해독(decoding) 어려움: 글자-소리 대응 규칙 적용 오류
- 읽기 유창성 부족: 느린 속도, 자주 막힘
- 읽기 이해 어려움: 해독은 되나 내용 이해가 안 됨

**쓰기 영역**
- 철자 오류 지속
- 문장 구조가 단순하고 반복적
- 작문 계획 및 조직화 어려움

**학업 영역**
- 교과서 텍스트 이해 어려움 (수학 문제 해석 포함)
- 학업 어휘 부족 (예: 나열하다, 분석하다, 비교하다)
- 수업 중 구어 지시 따르기 어려움

## 언어기반 학습장애 (LBLD)

읽기장애(난독증 포함)와 쓰기장애의 상당 부분은 언어, 특히 음운 처리 능력의 어려움에 기인한다.

### 음운 처리 능력의 세 가지 핵심 구성 요소

1. **음운 인식(Phonological Awareness)**: 단어 내 소리를 인식하고 조작하는 능력
   - 음절 분리: "사과" → [사] [과]
   - 음소 분리: "사" → [ㅅ] [ㅏ]
   - 음소 합성, 삭제, 대치
2. **음운 작업기억(Phonological Working Memory)**: 청각 정보를 단기 저장하는 능력
   - 비단어 따라 말하기 과제로 측정
3. **빠른 자동화 이름대기(RAN, Rapid Automatized Naming)**: 글자, 숫자, 색, 그림을 빠르게 이름 대는 능력
   - 읽기 유창성의 강력한 예측 변수

## 학령기 언어 평가 도구

### LSSC (Language Scale for School-aged Children, 학령기 아동 언어 검사)

- 대상: 만 7~12세
- 영역: 어휘(의미), 문법(구문·형태), 화용, 듣기 이해, 읽기
- 활용: 학령기 DLD 진단, 영역별 강약점 파악

### KOPLAC (Korean Protocol for Language Assessment for Children)

- 대상: 학령기 및 취학 전 후기
- 언어 능력을 다차원적으로 평가하는 프로토콜

### 이야기 평가 (Narrative Assessment)

- 그림 보고 이야기 구성 (예: 버스 이야기, Mayer 개구리 이야기)
- 이야기 구조 분석: 배경, 계기, 시도, 결과, 반응
- 결속 장치(접속어, 대명사 사용) 분석
- MCA(Main Concept Analysis): 핵심 개념 전달력 측정

### 비단어 따라 말하기 (Nonword Repetition)

- 음운 작업기억 측정
- DLD와 LBLD의 민감한 선별 지표

### 음운 인식 검사

- 음절 수 세기, 음절 분리, 음소 조작 과제
- 국내: 한국어 읽기 검사(KOLRA)의 음운 인식 하위 검사

### KOLRA (Korean Language-based Reading Assessment)

- 대상: 초등 1~6학년
- 하위 검사: 해독, 읽기 유창성, 읽기 이해, 어휘, 음운 인식, 빠른 자동화 이름대기

## 학령기 핵심 평가 영역

### 이야기(내러티브) 산출 평가

학령기 언어 능력에서 이야기 산출 능력은 학업 성취, 읽기 이해, 작문과 강하게 연결된다.

평가 방법:
- 그림 자극 이야기 산출: 6~8장 그림 카드 순서 보고 이야기 구성
- 이야기 재화: 들은 이야기를 다시 말하기

분석 항목:
- 이야기 구조 완성도 (배경, 계기, 행동, 결과, 반응)
- 발화 수, MLU (학령기에서도 복잡한 구문 발달 확인에 유효)
- 접속사 종류 및 빈도 (그리고, 그래서, 그런데, 왜냐하면)
- 내적 상태어 사용 (생각, 느낌, 믿음)
- 결속 장치 오류

### 읽기 평가

해독과 읽기 이해를 분리해 평가한다. 해독이 잘 되어도 이해가 안 될 수 있고, 반대의 경우도 있기 때문이다.

| 평가 내용 | 도구/방법 |
|---|---|
| 해독 | KOLRA 해독 하위검사, 실제 단어+비단어 |
| 읽기 유창성 | 1분 음독 정확도·속도 |
| 읽기 이해 | 문단 읽고 질문 답하기 |
| 음운 인식 | 음절/음소 분리·합성·조작 |
| RAN | 글자, 숫자, 색깔 이름 대기 |

### 작문 평가

- 그림 자극 이야기 쓰기 또는 특정 주제 작문
- 분석: 총 단어 수, 문장 수, 문장 복잡도, 철자 오류, 구두점, 주제 일관성

## 학령기 언어재활 중재 방향

### 이야기 중재 (Narrative Intervention)

이야기 구조 도식(story grammar) 가르치기:
- 배경(Setting), 계기(Initiating Event), 시도(Attempt), 결과(Consequence), 반응(Reaction)의 5요소
- 이야기 지도(Story Map) 시각 자료 활용
- 교육과정 내 이야기 텍스트와 연계

### 어휘 중재 (Vocabulary Intervention)

학업 어휘와 학과목 어휘 중심:
- 의미지도(Semantic Mapping) 기법
- 단어 연구(Word Study): 어원, 접두사, 접미사 분석
- 문맥 단서 사용 전략

대상 어휘 우선순위 (Beck 외, 3-tier model):
- Tier 1: 기본 일상 어휘 (먹다, 가다 등) — 대부분 학령기에 이미 습득
- Tier 2: 학업 어휘 (비교하다, 분석하다, 예측하다) — 임상에서 중점
- Tier 3: 교과 전문 어휘 (광합성, 대류, 소수) — 교과 연계

### 음운 인식 중재 (Phonological Awareness Intervention)

읽기 어려움이 있는 아동에게 핵심 중재:
- 음절 분리·합성: 박수치기, 블록 이동
- 음소 인식: 처음/마지막 소리 찾기, 운율 인식
- 글자-소리 대응(GPC, Grapheme-Phoneme Correspondence) 체계적 교수

근거: 체계적 음운 인식 중재가 읽기 해독에 효과적임은 메타분석으로 반복 확인됨 (Ehri 외, 2001)

### 실행기능 지원 (Executive Function Support)

DLD 아동은 계획, 조직화, 전환 등 실행기능에서도 어려움을 보이는 경우가 많다.
- 자기점검 체크리스트 활용
- 과제 분해(task decomposition) 전략
- 시각적 일정표와 순서도

### 학교 연계

- 담임교사 교육: DLD 아동의 학습 특성, 지원 방법
- 특수교육 협력: IEP(개별화교육프로그램) 연계
- 학습 자료 조정: 오디오 지원, 글자 크기 확대, 독해 지원

---

## 평가 도구 요약

| 도구 | 대상 | 영역 |
|---|---|---|
| LSSC | 만 7~12세 | 어휘, 문법, 화용, 듣기, 읽기 |
| KOLRA | 초등 1~6학년 | 읽기 종합 (해독, 유창성, 이해, 음운) |
| KOPLAC | 학령기 | 언어 다차원 평가 프로토콜 |
| 이야기 구성 | 학령기 | 내러티브 산출, 이야기 구조 |
| 비단어 따라말하기 | 학령기 | 음운 작업기억 |
| 이야기 듣기 후 재화 | 학령기 | 청각적 이해, 이야기 기억 |

## SOAP Note 작성 시 유의사항

### S

- 보호자 보고: 학교 성적, 읽기 속도, 받아쓰기 오류, 발표 회피
- 교사 보고: 수업 이해, 지시 따르기, 글쓰기 수준
- 아동 보고(가능하면): "어떤 게 제일 어려워요?"

### O

- LSSC 하위검사 점수 (표준점수, 백분위)
- KOLRA 하위검사 점수
- 이야기 산출 분석: 이야기 구조 점수, MLU, 결속장치 오류 수
- 비단어 따라말하기 정확도

### A

- 영역별 강약점 해석 (읽기 해독 vs 이해의 분리 여부)
- 음운 처리 능력 수준
- DLD 여부 및 학업 영향 범위
- 동반 어려움 유무 (주의력, 실행기능)

### P

- 목표 영역 우선순위 (읽기/어휘/이야기 중 선택)
- 주 2~3회 개별 치료 권장 여부
- 교실 연계 전략 협의 여부
- 가정 지원 방법 (책 읽어주기, 대화 확장)
