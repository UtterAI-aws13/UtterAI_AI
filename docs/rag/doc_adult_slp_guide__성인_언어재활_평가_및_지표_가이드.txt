# 성인 언어재활 평가 및 지표 가이드

## 개요

성인 언어재활은 아동과 달리 언어 발달이 아닌 후천적 손상 후 기능 회복과 보상 전략 습득을 목표로 한다. 주요 대상은 뇌졸중·외상에 의한 실어증, 외상성 뇌손상(TBI)의 인지-의사소통 장애, 퇴행성 질환(치매), 신경학적 손상에 의한 마비말장애·말실행증, 음성장애 등이다.

성인은 이미 언어 발달이 완성된 상태이므로 MLU 같은 발달 지표보다 기능적 의사소통, 담화 정보량, 영역별 수행 정확도가 핵심 지표가 된다.

---

## 주요 평가 영역

| 영역 | 대상 | 핵심 평가 내용 |
|---|---|---|
| 실어증 | 뇌졸중, 뇌손상 후 언어장애 | 유창성, 청각적 이해, 따라말하기, 이름대기, 읽기·쓰기 |
| 인지-의사소통 | TBI, 치매, 우반구 손상 | 주의, 기억, 집행기능, 담화 조직화, 화용 |
| 말운동 | 마비말장애, 말실행증 | 말 명료도, 조음 정확도, 말속도, 운율 |
| 음성장애 | 성대 질환, 기능적 음성장애 | 음질, 음도, 강도, 음성 피로 |
| 유창성 | 성인 말더듬 | 비유창성 빈도, 유형, 회피 행동 |

---

## 성인과 아동의 지표 차이

아동에서 MLU는 언어 발달 수준을 보여주는 핵심 지표이지만, 성인에서는 다른 이유로 쓰인다.

성인은 발화 길이보다 정보를 얼마나 정확하고 효율적으로 전달하는지가 중요하다. 같은 발화 길이라도 핵심 정보를 전달하는 발화와 핵심 없이 길기만 한 발화는 임상적 의미가 다르다.

성인 담화 분석에서는 CIU 계열 지표가 중심이 된다. 실어증 성인이 그림을 설명할 때 "어… 저기… 여자… 물… 넘쳐…"처럼 발화가 짧더라도 핵심 정보를 담고 있으면 의사소통이 가능하며, 반대로 말이 길어도 핵심 내용이 없으면 효율성이 낮다.

---

## 주요 지표: CIU 계열

### CIU (Correct Information Unit, 맥락적 정보 단위)

**정의**: 문맥상 명료하고, 주제나 과제에 적합한 정보를 제공하는 단어

**산출 지표**:
- 총 CIU 수: 전체 발화에서 CIU 개수
- %CIU: 총 단어 수 중 CIU 비율 (CIU 수 ÷ 총 단어 수 × 100)
- CIU/min (분당 CIU 수): 단위 시간당 정보 전달 효율성

**임상적 의미**:
- %CIU: 발화에서 정보를 담은 단어의 비율. 실어증 중증도에 따라 낮아진다.
- CIU/min: 말속도와 정보량을 동시에 반영. 재활 효과 측정에 유용.

**CIU 적용 예시**:
- 그림 설명 과제에서 "아, 저기 음… 물이 넘치고 있어요"에서 "물", "넘치고"가 과제에 적합한 정보를 담고 있으면 CIU로 계산
- 과제와 무관한 발화, 반복, 단순 감탄사는 CIU로 계산하지 않음

AphasiaBank는 담화 분석 도구로 Main Concept Analysis, Core Lexicon, CIU 계산, 자동 분석 등을 제공한다.

### Main Concept Analysis

구어 담화에서 핵심 개념 전달 여부를 평가한다. 과제별로 사전에 정의된 핵심 개념 목록과 대조해 아동이 전달한 개념의 수와 정확도를 측정한다.

## 실어증 평가

### 실어증 유형별 특성

| 유형 | 유창성 | 이해 | 따라말하기 | 이름대기 | 특징 |
|---|---|---|---|---|---|
| Broca 실어증 | 비유창 | 비교적 양호 | 어려움 | 어려움 | 전보식 발화, 문법형태소 생략 |
| Wernicke 실어증 | 유창 (내용 부족) | 어려움 | 어려움 | 어려움 | 착어, 신조어, 공허한 발화 |
| 전도성 실어증 | 유창 | 비교적 양호 | 심한 어려움 | 다소 어려움 | 자기수정 시도, 음운착어 |
| 전반성 실어증 | 비유창 | 어려움 | 어려움 | 어려움 | 중증. 모든 양식에서 심한 어려움 |
| 명칭 실어증 | 유창 | 양호 | 양호 | 어려움 | 단어 찾기 어려움이 두드러짐 |

### 오류 유형

- 의미착어(semantic paraphasia): 목표 단어와 의미적으로 관련된 단어로 대치 (예: 사과→배)
- 음운착어(phonemic paraphasia): 목표 단어와 음소가 일부 같은 단어로 대치 (예: 사과→사가)
- 신조어(neologism): 실제 존재하지 않는 단어 산출
- 우회표현(circumlocution): 목표 단어를 말하지 못하고 설명으로 대신함

### 주요 검사 도구

**PK-WAB (Western Aphasia Battery - Korean)**:
- 자발화, 청각적 이해, 따라말하기, 이름대기, 읽기, 쓰기, 구성 과제 포함
- 실어증 지수(AQ: Aphasia Quotient) 산출
- 실어증 유형 분류 가능

**K-BNT (Korean-Boston Naming Test)**:
- 이름대기 능력 심화 평가
- 단서 제공 후 반응(음소 단서, 의미 단서)으로 이름대기 처리 방식 파악 가능

## 인지-의사소통 장애

외상성 뇌손상(TBI), 치매, 우반구 손상에서 나타난다. 언어 능력 자체보다 언어 사용의 효율성과 맥락 적절성이 저하된다.

### 주요 평가 영역

- 주의(attention): 지속적 주의, 선택적 주의, 분리 주의
- 기억(memory): 작업기억, 새로운 학습, 회상
- 집행기능(executive function): 계획, 조직화, 문제해결, 억제
- 화용(pragmatics): 주제 유지, 대화 차례, 추론, 암시 이해
- 담화 조직화: 이야기 구성, 설명 담화, 정보 서열화

### 치매 단계별 언어 변화

- 경도: 단어 찾기 어려움, 담화에서 주제 이탈, 기억 기반 이야기 어려움
- 중등도: 반복적 발화, 이해 어려움 증가, 문장 완성 어려움
- 중증: 발화량 감소, 반향어, 무언증으로 진행 가능

### TBI 특성

외상 후 혼돈 및 기억상실 기간이 지나면 인지-의사소통 어려움이 두드러진다. 주의력 저하가 담화와 대화 능력에 광범위하게 영향을 미친다. ASHA는 TBI 평가 목적을 말·언어·인지-의사소통·삼킴·청각 영역의 강점과 요구 파악으로 설명한다.

## 말운동 장애 (마비말장애, 말실행증)

### 마비말장애 (Dysarthria)

신경학적 손상으로 인한 말 산출 근육 기능 저하. 호흡, 발성, 조음, 공명, 운율에 영향을 미친다.

유형별 특성:
- 이완형: 과비성, 기식음성, 짧은 호흡군
- 경직형: 낮은 음도, 긴장된 음성, 불규칙한 조음 붕괴
- 운동실조형: 불규칙한 조음 오류, 과장된 강세, 불규칙한 말속도
- 혼합형: 여러 유형의 특성이 혼재

주요 지표: 말 명료도(%), 말속도(WPM), 조음 정확도

### 말실행증 (Apraxia of Speech)

운동 계획·프로그래밍 장애. 근육 약화 없이 말 산출이 어렵다.

특성: 자음과 모음 모두 오류, 시행착오적 탐색, 불일관적 오류, 자동적 발화가 의도적 발화보다 쉬움, 말속도가 느림

중재: 집중적이고 반복적인 음소·단어 수준 훈련, 점진적 복잡도 증가

## 성인 정량 지표 정리

| 치료 목표 | 정량 지표 | 의미 |
|---|---|---|
| 이름대기/단어 찾기 | 정반응률 %, 단서 후 정반응률, 반응시간 | 단어 산출 정확도 변화 |
| 청각적 이해 | 지시 따르기 정답률, 문장 이해 점수 | 말 이해 향상 |
| 따라말하기 | 단어/구/문장 반복 정확도 | 반복 산출 능력 |
| 담화/자발화 | %CIU, CIU/min, 총 단어 수, 말속도(WPM) | 정보 전달 효율성 |
| 기능적 의사소통 | FCM 점수, ASHA-FACS | 일상 의사소통 실제 기능 |
| 인지-의사소통 | 과제별 수행률, 화용 척도 | 인지 기반 의사소통 기능 |
| 말운동/음성 | 명료도 %, 말속도, 조음 정확도 | 말 명료도와 효율성 |

## 정성 지표: 수치로 보이지 않는 것

정량 지표와 함께 임상 관찰과 보호자 보고를 통한 정성 평가가 필수적이다. 수치가 향상되어도 일상 대화가 나아지지 않을 수 있고, 수치가 낮아도 보상 전략을 통해 기능적 의사소통이 가능할 수 있다.

| 정성적으로 보는 것 | 예시 |
|---|---|
| 오류 양상 변화 | 의미착어 감소, 음운착어로 유형 변화 |
| 전략 사용 | 제스처, 쓰기, 첫소리 단서 요청, AAC 활용 |
| 대화 능력 | 주제 유지, 수리 전략, 자기 수정 |
| 기능 변화 | 가족과 통화, 병원에서 요구 표현 |
| 보호자 반응 | 단서 제공 빈도 변화, 의사소통 부담 감소 |
| 참여·삶의 질 | 모임 참여, 자신감, 좌절감 감소 |

---

## 치료 목표 예시

정량 목표:
- "그림 설명 과제에서 %CIU를 35%에서 55% 이상으로 향상한다"
- "대면이름대기 과제에서 무단서 정반응률을 40%에서 70% 이상으로 향상한다"
- "2단계 구어 지시 따르기 정확도를 60%에서 85% 이상으로 향상한다"

정성 목표:
- "대화 상황에서 단어 찾기 어려울 때 제스처·쓰기 보완 전략을 자발적으로 사용한다"
- "10분 대화에서 주제 이탈 없이 일관된 주제를 유지한다"

ASHA의 NOMS(National Outcomes Measurement System)는 치료 시작과 종결 시점에 기능적 의사소통 측정치(FCMs)를 기록해 전후 점수 차이로 효과를 확인한다.

## SOAP Note 예시 (성인 실어증)

**S**: 보호자에 따르면 지난 주보다 가족과의 전화 의사소통이 다소 쉬워졌다고 보고함.

**O**: 대면이름대기 정반응률 72% (이전 45%). 그림 설명 과제 %CIU 54% (이전 38%). 단어가 막힐 때 제스처와 쓰기를 자발적으로 사용하기 시작함.

**A**: 이름대기와 담화 정보량 모두 향상. 보상 전략 사용이 증가해 일상 의사소통 기능 개선이 관찰됨. 음운착어 빈도는 여전히 높아 지속적인 단어 산출 훈련 필요.

**P**: 음소 단서 의존도 감소를 목표로 이름대기 집중 훈련 지속. 보호자에게 대화 중 자연스럽게 기다려주기 전략 교육.

## SOAP Note 작성 원칙

SOAP Note는 세션 기록 양식으로 Subjective(주관적), Objective(객관적), Assessment(해석), Plan(계획) 네 영역으로 구성된다. 아동·성인 모두 동일한 구조를 사용하며, 기록 대상과 지표만 달라진다.

### 영역별 작성 기준

**S (Subjective, 주관적 정보)**
치료사 관찰 또는 보호자 보고를 서술한다. 수치가 아닌 서술 정보.
예: "보호자에 따르면 지난 주보다 가족과의 대화가 다소 쉬워졌다고 보고함."

**O (Objective, 객관적 정보)**
측정 가능한 수치와 관찰 가능한 행동을 기록한다.
예: "대면이름대기 정반응률 72%. 그림 설명 과제 %CIU 54%. 단어가 막힐 때 제스처와 쓰기를 자발적으로 사용하기 시작함."

**A (Assessment, 해석)**
수치와 관찰을 임상 기준에 비추어 해석한다.
예: "이름대기와 담화 정보량 모두 향상. 보상 전략 사용이 증가해 일상 의사소통 기능 개선이 관찰됨. 음운착어 빈도는 여전히 높아 지속적인 단어 산출 훈련 필요."

**P (Plan, 계획)**
다음 회기 목표와 활동 방향을 기술한다.
예: "음소 단서 의존도 감소를 목표로 이름대기 집중 훈련 지속. 보호자에게 기다려주기 전략 교육 예정."

### 작성 주의사항

단정적 표현 금지: "진단한다", "결론이다" 사용 금지
권장 표현: "소견이다", "보임", "가능성이 있음", "것으로 보임", "확인이 필요함"
수치는 제공된 데이터를 그대로 인용하고 임의로 만들지 않는다.
치료사가 검토하는 초안임을 명시한다.

## RAG 활용 맥락

- 세션에서 실어증 대상 성인이 확인될 때 유형별 특성과 평가 방향 제공
- %CIU 수치가 낮을 때 담화 중재 전략과 목표 제안
- 마비말장애 대상의 경우 말 명료도 훈련 접근법 안내
- TBI 대상 인지-의사소통 어려움이 언급될 때 주의·기억·집행기능 평가 필요성 안내
- SOAP 작성 시 성인 대상 정량·정성 지표 통합 기록 방법 제안
