# 한국어 형태통사 임상 가이드

## 왜 한국어는 형태소 분석이 중요한가

영어권 언어표본분석과 달리 한국어는 조사와 어미가 의미·문법 관계를 결정하는 교착어다. 같은 어휘를 사용하더라도 조사와 어미의 사용 여부에 따라 문법성과 의미관계가 완전히 달라진다.

예시 비교:
- "토끼가 당근을 먹어요" → 어절 3개, 형태소는 '토끼+가+당근+을+먹+어+요'로 분절
- "토끼 먹어" → 어절 2개이지만 주격조사(-가), 목적격조사(-를), 종결어미(-요)가 모두 누락

단순히 발화 길이나 낱말 수만 보면 두 발화의 차이가 작아 보이지만, 임상적으로는 조사 사용, 문장 확장, 의미관계 표현 측면에서 큰 차이가 있다.

따라서 한국어 아동 발화 분석은 어절, 단어, 형태소를 구분해 분석해야 하며, 단순 어절 수나 발화 길이만으로는 언어 발달 수준을 정확히 파악하기 어렵다.

## 분석 단위 비교

| 단위 | 설명 | "토끼가 먹어요" 예시 |
|---|---|---|
| 어절 | 띄어쓰기 단위 | 2어절 |
| 낱말/단어 | 조사를 제외한 어휘 단위 | 토끼, 먹다 (2단어) |
| 형태소 | 최소 의미 단위 | 토끼 + 가 + 먹 + 어 + 요 (5형태소) |

MLU-m(형태소 단위)이 MLU-w(낱말 단위)보다 한국어 아동의 문법 발달을 더 민감하게 반영한다. 형태소 분석을 통해 조사와 어미 사용 여부를 직접 확인할 수 있기 때문이다.

## 주요 문법형태소 유형

### 격조사

| 조사 | 기능 | 오류 예시 |
|---|---|---|
| -이/-가 | 주격 (행위자, 주어 표시) | "토끼 먹어" (주격조사 누락) |
| -을/-를 | 목적격 (대상 표시) | "사과 먹어" (목적격조사 누락) |
| -에 | 처소격 (장소 표시) | "집 가요" (처소격 누락) |
| -에서 | 출발점 또는 장소 | "학교 왔어" (출발점 표시 누락) |
| -의 | 관형격 (소유) | "엄마 가방" (관형격 누락) |
| -이랑/-과/-하고 | 공동격 (함께) | "친구 놀았어" (공동격 누락) |

### 어미

| 어미 유형 | 예시 | 발달적 의미 |
|---|---|---|
| 종결어미 | -아/어, -아요/어요, -다 | 문장 완성도 |
| 연결어미 | -고, -서, -는데, -면 | 절 연결 능력, 문장 복잡도 |
| 전성어미 | -는, -ㄴ, -ㄹ (관형형), -기, -음 (명사형) | 복문 산출 능력 |

### 오류 패턴 분류

1. 누락(omission): 필수 형태소가 빠짐. 가장 흔한 오류 유형.
   예: "엄마 밥 줘" → 주격(-가)과 목적격(-을) 모두 누락

2. 대치(substitution): 목표 형태소 대신 다른 형태소 사용.
   예: "학교에서 왔어" → "학교에 왔어" (출발점 -에서 대신 처소 -에 사용)

3. 첨가(addition): 불필요한 형태소 추가.

## 임상적 해석 기준

### 발달 순서

일반적으로 국내 연구에서 보고된 조사 발달 순서는 다음과 같다.
- 일찍 발달: 주격(-이/-가), 목적격(-을/-를), 처소격(-에)
- 이후 발달: 도구격(-로/-으로), 공동격(-이랑/-하고), 관형격(-의)
- 늦게 발달: 주제격(-은/-는) 기능 이해, 연결어미 다양화

### 단순언어발달지연 아동의 특징

- 2~3세 수준의 짧은 발화가 지속됨
- 격조사 누락이 반복됨
- 어미 다양성이 낮고 "-아/어" 종결 위주로 고착됨
- 연결어미(-고, -서)가 제한적

### 발달성 언어장애(DLD) 아동의 특징

- 연령에 비해 문법형태소 오류가 지속됨
- 종결어미는 사용하지만 연결어미, 전성어미 발달이 늦음
- 복문 산출이 어렵고 단문 위주의 발화가 지속됨

## 중재 목표 설정 원칙

1. 현재 자발화에서 50~90% 정확도로 사용하는 형태소를 우선 목표로 선정한다. 전혀 사용하지 않는 형태소보다 이미 나타나지만 불안정한 형태소가 중재 효과가 높다.

2. 발달 순서를 고려해 이른 발달 조사부터 목표로 한다.

3. 가정과 자연스러운 상황에서 일반화를 목표로 한다.

중재 목표 예시:
- "그림 설명 상황에서 주격조사(-이/-가)를 80% 이상 정확하게 산출한다"
- "2어문에서 목적격조사(-을/-를)를 포함한 문장을 자발적으로 산출한다"
- "행위 그림 설명 시 'X가 Y해요' 문장 형식으로 산출한다"

## 부모 지도 연계

치료사가 자연스러운 언어 환경을 만들기 위해 부모에게 제안할 수 있는 전략:
- 동작 그림·동물 놀이에서 "토끼가 먹어요", "강아지가 자요" 형태로 모델링
- 아동 발화에서 조사가 누락된 경우 반복 확장(recasting): 아동이 "토끼 먹어"라고 하면 "토끼가 먹어요!"로 자연스럽게 되받아 말하기
- 지적하거나 교정하지 않고 올바른 형태를 반복 제공하는 방식

## RAG 활용 맥락

- 세션 분석에서 주격조사 누락이 반복되면 중재 목표 후보와 부모 전략 추천
- 문법형태소 오류율이 높을 때 발달 수준에 맞는 목표 문법형태소 제안
- MLU-m과 MLU-w 차이가 클 때 형태소 발달 지연 가능성 안내
