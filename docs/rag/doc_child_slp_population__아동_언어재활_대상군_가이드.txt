# 아동 언어재활 대상군 가이드

## 개요

국내 언어재활은 단일한 "말 늦음" 아동만을 다루지 않는다. 영유아 언어발달, 취학 전 수용·표현언어, 어휘, 구문·의미, 조음·음운, 유창성, 사회적 의사소통, 보완대체의사소통, 학령기 언어 등 폭넓은 영역을 포함한다.

ASHA는 아동의 구어 언어장애를 음운론, 형태론, 구문론, 의미론, 화용론의 다섯 언어 영역에서 듣기·말하기 기술의 습득·사용에 지속적인 어려움이 있는 상태로 정의한다.

---

## 1. 단순 언어발달지연 / 발달성 언어장애(DLD)

### 특성

단순 언어발달지연 아동은 인지, 청력, 신경학적 문제 없이 언어 발달이 늦는 경우다. 발달성 언어장애(Developmental Language Disorder, DLD)는 언어 문제가 지속되고 다른 발달 영역과 설명되지 않는 경우를 가리킨다.

주요 특징:
- 수용언어, 표현언어, 어휘, 문장 길이, 문법형태소, 대화 능력, 이야기 능력 전반에 걸쳐 어려움
- 동일 연령 아동에 비해 MLU, NDW, TTR이 낮음
- 격조사·어미 같은 문법형태소 오류가 반복됨
- 복문 산출이 어렵고 단문 위주의 발화가 지속됨

### 주요 평가 지표

- 표준화 검사: PRES(2~6세), SELSI(영유아), REVT(어휘)
- 언어표본분석: MLU-m, MLU-w, LLU, NTW, NDW, TTR, 문법형태소 오류
- K-ALAS로 자동화 분석 가능

### 중재 방향

- 문장 길이 확장 (2어문→3어문→복문)
- 어휘 다양성 증가 (의미 범주별 어휘 목표)
- 격조사·어미 등 문법형태소 산출
- 의미관계 확장 (행위자-행위, 행위자-대상-행위)
- 질문 이해·응답 능력
- 이야기 구성 (이야기 문법 구조)

## 2. 조음·음운장애 / 말소리장애

### 특성

특정 말소리를 생략, 대치, 왜곡하거나 나이에 비해 부정확한 발음 패턴이 지속되는 경우다. ASHA는 말소리장애 아동의 경우 특정 말소리 오류 때문에 언어평가 수행이 어렵고, 말 명료도가 낮으면 언어표본 수집 및 분석이 어려울 수 있다고 설명한다.

주요 오류 유형:
- 대치: 목표 음소를 다른 음소로 바꿈 (예: /ㅅ/ → /ㄷ/)
- 생략: 음소를 빠뜨림 (예: 어말 자음 생략)
- 왜곡: 음소를 부정확하게 산출

### 주요 평가 지표

- PCC (자음정확도)
- 음운변동 분석 (어떤 패턴의 오류가 반복되는지)
- U-TAP2 (만 2세 6개월~7세), APAC

U-TAP2 표준화 연구는 서울·경기·충청·경상·전라·제주 아동 616명을 대상으로 타당도와 신뢰도를 검토했다. 단어·문장 수준에서 자음정확도와 주요 측정치 간 구인타당도, 연령별 자음정확도 차이, 검사-재검사 신뢰도가 확인되었다.

2024년 연구에서 한국어 말소리 자동분석도구(KSAT)와 임상가 수동분석을 비교한 결과 오류패턴분석 일치도가 93.63%로 보고되었다.

### 중재 방향

- 목표 음소 선정: PCC, 오류 패턴, 자극반응도, 발달 순서 고려
- 음소 수준 → 음절 → 단어 → 문장 → 자발화 순서로 확장
- 최소대립쌍 훈련, 주기법(Cycles Approach) 등 음운론적 접근

## 3. 자폐스펙트럼장애(ASD) 아동

### 특성

자폐스펙트럼장애 아동의 언어치료는 말의 양보다 사회적 의사소통, 공동주의, 상호작용, 대화 차례, 요구하기, 반응하기, 정서·행동과의 연계를 함께 다룬다.

서울대학교어린이병원 자폐스펙트럼클리닉은 조기 진단·개입, 보호자 상담, 특수학교·사회복지 연계, 모아애착반·학교준비반·사회기술훈련 등 발달 단계별 집단 프로그램을 운영한다.

주요 특징:
- 사회적 의사소통 어려움 (공동주의, 상호작용 참여)
- 발화량이 있더라도 화용 기능이 제한적 (요구 위주, 공유·반응 기능 부족)
- Turn-taking 어려움 (자발 발화 시작 감소, 주제 유지 어려움)
- 반향어(echolalia) 가능

### 주요 평가 지표

- Turn-taking 빈도 및 유형 (자발 발화 비율, 반응 발화 비율)
- 화용 기능 분류 (요구, 응답, 공유, 거절, 언급)
- 공동주의 행동 관찰
- 언어표본에서 반향어 비율

### 효과적인 중재 전략

강화된 환경 중심 중재(Enhanced Milieu Teaching) 메타분석 결과:
- 공동주의 증진: 흥미로운 자료 사용, 도움이 필요한 상황 유발 전략
- 의사소통 기능 다양화: 모델링, 시간지연 전략
- 차례 주고받기 증진: 아동 행동·발화에 반응하기, 선 반응 요구-후 시범 전략

영국 NICE 지침은 자폐 아동·청소년 지원을 지역 기반 다학제 팀(건강, 정신건강, 교육, 사회복지 포함)이 담당하도록 권고한다.

## 4. 학령기 언어장애 / 이야기·화용·읽기 언어

### 특성

취학 전 아동뿐 아니라 학령기 아동도 언어치료 대상이다. ASHA는 구어 언어장애가 읽기·쓰기 학습에 영향을 줄 수 있으며, 특히 설명 담화 같은 고차 언어기술에서 어려움을 보일 수 있다고 설명한다.

주요 어려움:
- 이야기 구성: 이야기 문법(발단, 계기, 시도, 결과, 반응) 이해 및 산출
- 설명 담화: 원인-결과, 비교-대조, 순서 설명
- 화용언어: 비유, 유머, 암시 이해, 맥락에 맞는 언어 사용
- 학업 언어: 교과서 이해, 학습 내용 설명, 쓰기

### 주요 평가 도구

- LSSC (학령기 아동 언어 검사)
- KOPLAC (한국 아동 메타-화용 언어검사)
- 언어문제해결력검사

### 중재 방향

- 이야기 구조 훈련 (이야기 문법 명시적 교수)
- 추론 훈련 (원인, 결과, 예측, 평가적 추론)
- 학업 어휘 및 설명 담화 훈련
- 읽기-언어 연계 중재

## 대상군별 주요 평가·중재 포인트 요약

| 대상군 | 핵심 평가 영역 | 핵심 중재 목표 |
|---|---|---|
| 단순 언어발달지연/DLD | MLU, 문법형태소, 어휘, PRES | 문장 확장, 조사·어미 산출, 어휘 다양성 |
| 조음·음운장애 | PCC, 음운변동, U-TAP2/APAC | 목표 음소 정확도, 음운 패턴 수정 |
| ASD | Turn-taking, 화용 기능, 공동주의 | 사회적 의사소통, 의사소통 기능 다양화 |
| 학령기 언어장애 | 이야기 구성, 담화, 화용, LSSC | 이야기 문법, 추론, 학업 언어 |
| 영유아 (0~2세) | SELSI, MCDI-K, 초기 의사소통 | 초기 의사소통 기능, 어휘 시작 |

## RAG 활용 맥락

- 세션 정보에서 대상군이 ASD로 확인되면 화용 중재 전략과 Turn-taking 목표 우선 추천
- 조음 오류가 STT 결과에 보정되었을 가능성이 있을 때 수동 확인 필요성 안내
- 학령기 아동에서 이야기 산출이 짧을 때 이야기 문법 중재 필요성 제안
