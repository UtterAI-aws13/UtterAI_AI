# TBI 인지-의사소통 평가 가이드

## 범위
외상성 뇌손상(TBI) 이후 의사소통은 주의, 기억, 처리속도, 집행기능, 자기인식, 사회적 인지의 영향을 받는다. 단어·문장 검사에서 정상 범위여도 실제 대화, 학업, 직장 복귀에서 어려움이 나타날 수 있다.

## 급성기 안전 확인
각성 수준, 섬망, 의학적 안정성, 청력·시력, 통증, 약물, 피로를 먼저 확인한다. 갑작스러운 신경학적 악화, 새 연하 문제, 심한 두통·의식 변화는 즉시 의료팀에 알린다.

## 평가 영역
주의는 선택·지속·교대·분할 주의를 기능 과제에서 본다. 기억은 즉각 회상만이 아니라 새 정보 학습, 지연 회상, 미래 기억, 외부 도구 사용을 평가한다.

집행기능은 목표 설정, 계획, 시작, 순서화, 문제 해결, 오류 감지와 수정으로 나눈다. 사회적 의사소통은 주제 유지, 말의 양 조절, 비유·간접 의미, 감정과 관점 추론, 의사소통 실패 수정 능력을 본다.

자기인식은 수행 예측과 실제 수행을 비교하고, 본인·가족·치료사의 보고 차이를 기록한다. 표준화 검사와 실제 일정 관리, 약 복용 설명, 이메일 작성, 회의 참여 같은 개인별 과제를 결합한다.

## 중재
전략 기반 중재는 목표-계획-실행-검토의 반복, 자기말하기, 체크리스트, 오류 예측과 피드백을 사용한다. 외부 보상 전략은 일정표, 알람, 메모, 환경 표지, 작업 단계표를 실제 생활에 배치한다.

새 학습이 어려우면 정보량을 줄이고 반복·간격 회상·일관된 루틴을 사용한다. 가족과 직장·학교 파트너에게 짧은 지시, 처리 시간, 이해 확인, 환경 방해 감소 방법을 교육한다.

## 목표 예시
“대상자는 직장 복귀 모의 과제에서 체크리스트를 사용해 5단계 업무를 순서 오류 1회 이하로 완료하고, 종료 후 오류를 독립적으로 검토한다.”

## 보고 원칙
검사 점수와 기능적 영향, 필요한 지원 수준을 분리한다. TBI만으로 모든 행동 변화를 설명하지 않으며 우울·불안·수면·통증과 다른 전문 영역을 협진한다.
