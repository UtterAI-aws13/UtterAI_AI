# 반응 지연 시간 해석 가이드

반응 지연 시간(Response Latency)은 치료사 발화 종료 후 아동 발화 시작까지의 시간입니다.

## 일반 기준

0.5초~1.5초: 정상 범위, 대화 흐름이 원활함
1.5초~2.5초: 경미한 지연, 처리 시간이 필요한 것으로 보임
2.5초 이상: 주의 필요, 처리 속도, 이해력, 집중도 확인 필요
0.5초 미만: 충동적 반응 또는 단순 반향어 가능성

## 반응 지연이 길 경우 함께 확인할 사항

1. 질문의 복잡도: 개방형 질문은 폐쇄형보다 반응 시간이 길어질 수 있음
2. 주의 집중도: 과제에 집중하지 못할 경우 지연 증가
3. 어휘 인출 어려움: 적절한 단어를 찾는 데 시간이 걸리는 경우
4. 청각 처리 속도: 말소리를 처리하는 속도가 낮을 경우

평균 반응 지연이 2.5초를 초과하는 경우 다음 회기에서 과제 구조나 질문 방식 조정을 고려합니다.
