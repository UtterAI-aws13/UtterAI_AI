# 임상 리포트 단정·진단 표현 금지 규칙

## 원칙

UtterAI가 생성하는 SOAP Note 초안은 언어재활사가 검토하는 참고 자료다. AI가 생성한 리포트에서 진단을 확정하거나 단정적 임상 판단을 내리는 표현은 사용하지 않는다.

## 절대 사용 금지 표현

### 진단 확정

금지: "~장애가 있다", "~장애로 진단된다", "~장애이다"
금지: "단순언어발달지연이다", "발달성 언어장애이다", "실어증이다"
금지: "자폐스펙트럼장애이다", "조음장애이다", "말더듬이다"

이유: 공식 진단은 자격을 갖춘 임상가가 표준화 검사, 면담, 종합 평가를 통해 내린다. AI 리포트는 진단 권한이 없다.

### 수치 기반 단정

금지: "MLU 2.3으로 언어발달지연이 확인된다"
금지: "PCC 65%이므로 조음장애다"
금지: "TTR 0.4로 어휘 장애가 있다"

이유: 단일 수치로 장애를 확정할 수 없다. 수치는 해석의 출발점이지 결론이 아니다.

### 예후 단정

금지: "치료 효과가 없을 것이다"
금지: "정상 발달이 불가능하다"
금지: "반드시 호전될 것이다"

이유: 예후는 다양한 변수(치료 빈도, 가정 지원, 동반 상태)에 따라 달라지며, AI가 단정할 수 없다.

### 보호자·내담자 평가

금지: "보호자가 협조적이지 않다"
금지: "아동이 치료 의지가 없다"

이유: 관계적 판단은 임상가가 직접 해야 한다.

## 권장 표현 (대체 표현)

| 금지 표현 | 권장 표현 |
|---|---|
| ~장애가 있다 | ~어려움이 관찰된다, ~소견이 있다, ~가능성이 있다 |
| ~로 진단된다 | 추가 평가가 권장된다 |
| ~이 확인된다 | ~것으로 보인다, ~시사된다 |
| 반드시 ~해야 한다 | ~을 고려할 수 있다, ~을 검토할 것을 권장한다 |
| ~는 정상이다 | 현재 회기에서 ~는 연령 기대 범위 내로 나타났다 |
| 치료가 필요하다 | 지속적인 언어재활 서비스가 도움이 될 것으로 보인다 |

## 수치 인용 원칙

수치는 제공된 세션 데이터를 그대로 인용하고 임의로 만들지 않는다.

올바른 예: "이번 회기 MLU-m은 2.3으로, 만 4세 기대 범위(3.0~4.5) 하단에 해당하는 것으로 보인다."
잘못된 예: "MLU 2.3은 매우 낮은 수준으로 심각한 언어발달지연을 나타낸다."

## 근거 부족 시 처리

세션 데이터나 참고 문헌이 충분하지 않아 판단이 어려운 경우, 없는 근거를 만들거나 추정으로 채우지 않는다.

권장: "현재 제공된 데이터만으로는 판단이 어려우며, 추가 평가가 필요하다."

## 초안 명시 의무

리포트 상단 또는 하단에 다음 문구를 포함한다.

"본 SOAP Note는 AI가 생성한 초안으로, 담당 언어재활사의 검토·수정 후 사용해야 합니다."

## 검토 권고 항목

다음 항목이 포함된 경우 언어재활사가 반드시 검토한다.

- 진단명이 포함된 문장
- 연령 규준 비교 해석
- 중재 목표 제안
- 보호자 교육 내용
- 의뢰(referral) 권고
